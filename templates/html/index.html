WiWA CMS

The Pythonic CMS Framework

Pythonで、Webサイトを思いのままに。
URLとコードが1対1でつながる、
シンプルでパワフルなPython製CMSフレームワーク。

WiWA CMSの特徴

  • Python First

    すべてがPythonで書ける。
    学びやすく、拡張しやすい設計。

  • URL = Code

    URLとコードが1対1でつながる。
    ルーティングの概念をシンプルに。

  • No Magic.

    魔法に頼らない、明示的な設計。
    コードがそのままドキュメント。

  • Light & Fast

    軽量で高速。
    必要なものだけを、あなたの手に。

SIMPLE IS POWERFUL.

Write Python,
That's it.

WiWA CMSでは、URLごとにPython関数を定義し、テンプレートを送るだけ。

シンプルな仕組みで、強力なWebサイトを構築できます。

SHOWCASE

作れるもの

コーポレートサイト、ブログ、ドキュメントサイト、ポートフォリオまで、シンプルな構成で柔軟に対応。

  • コーポレートサイト

    企業の顔となるサイトも、WiWA CMSならシンプルに構築可能。

  • ブログサイト

    シンプルなブログから、カスタム投稿タイプを使った複雑なサイトまで。
    WiWA CMSなら、あなたのアイデアを形にできます。

  • ドキュメントサイト

    プロジェクトのドキュメントサイトも、WiWA CMSなら簡単に構築可能。
    Markdownサポートも充実。

  • ポートフォリオサイト

    クリエイターの作品を魅力的に見せるポートフォリオサイトも、WiWA CMSならシンプルに構築可能。

さあ、PythonicなCMSを始めよう。

シンプルな一歩が、あなたのアイディアをカタチにします。
